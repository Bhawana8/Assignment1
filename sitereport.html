Site Report

Development Experience

Starting to learn web development has been challenging for me.HTML and CSS were all new and confusing for me.But as I practiced, things began to make sense. I learned how to create simple web pages with HTML and make them look nice with CSS. Writing code was exciting, tricky and difficult for me. Creating my first web page with HTML felt like a big accomplishment. Adding CSS allowed me to play with colors, fonts,and layouts to make the page look better. Designing the website was fun but required a lot of thought. choosing the right colors and fonts was important to make sure that the site looked good and was easy to read.

Reflective Discussion

Throughout the term, I learned more and more about web development. Each lesson built on what i had learned before. The early lessons focused on the basics, and later lessons introduced more advanced topics. Overall, I tried to write clean and efficient code. I learned about responsive design to ensure the site looked good on various devices, which involveed using media quires and flexible layouts. Design-wise, I aimed for simple and consistent, with a clean layout and nice color scheme.

Design Decisions

Chossing fonts and colors was a crucial part of the design process. I opted for a sans-serif font for its readability and modern look. In conclusion, The journey of learning web development has been great experience. There wew ups and downs, but each challenge taught me something new and made me better at coding and design
